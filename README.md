# Alocados Coin Exchange

## 설치 방법

1. 리포지토리를 클론합니다.

```bash
git clone https://github.com/gparkkii/alocados-coin-exchange.git
```

2. 프로젝트 디렉토리로 이동합니다.

```bash
cd alocados-coin-exchange
```

3. yarn으로 패키지들을 설치합니다.

```bash
yarn install
```

## 앱 실행

```bash
yarn start
```

 http://localhost:3000 에서 실행됩니다.
